## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
## Manufacturer Tests
To assist in testing different manufactures, mock data has been provided in `test/mock-data` directory that can be used to create new manufactures
Supported manufactures: `Samsung, Lenovo and Huawei`
